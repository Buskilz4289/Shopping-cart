# 📝 רשימת קניות - Shopping List App

אפליקציית רשימת קניות פשוטה ונוחה בעברית עם תמיכה מלאה ב-RTL.

## ✨ תכונות

### תכונות בסיסיות
- ✅ **תמיכה מלאה בעברית ו-RTL** - כל הממשק בעברית עם כיוון מימין לשמאל
- 📱 **רספונסיבי** - עובד מצוין על מחשב, טאבלט וטלפון
- 🌙 **מצב כהה** - אפשרות לעבור בין מצב בהיר לכהה
- 💾 **שמירה אוטומטית** - כל הנתונים נשמרים ב-LocalStorage
- 📦 **PWA** - ניתן להתקין כאפליקציה על המכשיר עם תמיכה offline
- 🎨 **עיצוב מודרני** - ממשק נקי ונוח לשימוש

### תכונות מתקדמות
- 🛒 **מצב קניות** - ממשק מפושט עם טקסט גדול לקניות נוחות
- ⭐ **מועדפים** - שמירת מוצרים קבועים לשימוש חוזר
- 🔄 **מוצרים חוזרים** - זיהוי אוטומטי של מוצרים שמופיעים לעיתים קרובות
- 📊 **סיכום חכם** - סטטיסטיקות בזמן אמת עם פילוח לפי קטגוריות
- 🔍 **השלמה אוטומטית** - הצעות בעת הקלדה ממועדפים, מוצרים חוזרים והיסטוריה
- 🧹 **ניקוי כפילויות** - זיהוי ומיזוג אוטומטי של פריטים כפולים
- 🔗 **שיתוף רשימות** - שיתוף רשימות עם משתמשים אחרים עם סנכרון בזמן אמת
- 📚 **היסטוריית קניות** - שמירה אוטומטית של רשימות קודמות
- 👆 **פעולות מגע** - החלקה ימינה לסמן כנקנה, שמאלה למחוק, לחיצה ארוכה למועדף
- 📳 **משוב טקטילי** - רטט בעת פעולות (במכשירים תומכים)

## 🚀 איך להריץ את האפליקציה

### שיטה 1: פתיחה ישירה (הכי פשוט)

1. פתח את הקובץ `index.html` בדפדפן
2. זה הכל! האפליקציה תעבוד מיד

### שיטה 2: שרת מקומי (מומלץ ל-PWA)

#### עם Python:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

ואז פתח בדפדפן: `http://localhost:8000`

#### עם Node.js (http-server):

```bash
# התקן http-server (אם עדיין לא מותקן)
npm install -g http-server

# הרץ שרת
http-server -p 8000
```

ואז פתח בדפדפן: `http://localhost:8000`

#### עם PHP:

```bash
php -S localhost:8000
```

ואז פתח בדפדפן: `http://localhost:8000`

## 📋 שימוש באפליקציה

### הוספת פריט חדש:

1. מלא את שם המוצר (חובה) - תתקבל השלמה אוטומטית ממועדפים והיסטוריה
2. אופציונלי: הוסף כמות וקטגוריה
3. לחץ על "הוסף לרשימה"

### סימון פריט כנקנה:

- לחץ על התיבה לצד הפריט
- הפריט יעבור אוטומטית לתחתית הרשימה
- ניתן לבטל את הסימון בכל עת
- **במובייל**: החלקה ימינה על הפריט

### מחיקת פריט:

- לחץ על כפתור "מחק" לצד הפריט
- תתבקש לאשר את המחיקה
- **במובייל**: החלקה שמאלה על הפריט

### מועדפים:

- לחץ על הכוכב ⭐ לצד הפריט כדי לסמן כמועדף
- **במובייל**: לחיצה ארוכה על הפריט
- מועדפים נשמרים גם אם הפריט נמחק מהרשימה

### מצב קניות:

- לחץ על כפתור 🛒 בכותרת
- ממשק מפושט עם טקסט גדול לקניות נוחות
- אופטימלי לשימוש ביד אחת במובייל

### סיכום חכם:

- מוצג אוטומטית בראש הדף
- מציג: סה"כ פריטים, נקנו, נותרו
- לחץ על "הצג לפי קטגוריה" לפילוח מפורט

### מוצרים חוזרים:

- האפליקציה מזהה אוטומטית מוצרים שמופיעים לעיתים קרובות בהיסטוריה
- מוצגים הצעות בראש הדף
- לחץ על מוצר כדי להוסיף אותו לרשימה

### השלמה אוטומטית:

- בעת הקלדת שם מוצר, תופיע רשימת הצעות
- הצעות ממועדפים (⭐), מוצרים חוזרים (🔄), והיסטוריה (📚)
- לחץ על הצעה או השתמש בחצים למעלה/מטה

### ניקוי כפילויות:

- לחץ על "נקה כפילויות" כאשר יש כפילויות
- האפליקציה תזהה ותמזג פריטים דומים

### שיתוף רשימות:

1. לחץ על "🔗 שתף" בראש הרשימה
2. העתק את מזהה הרשימה
3. שתף עם אחרים
4. הם יכולים להצטרף על ידי הכנסת המזהה
5. שינויים מסתנכרנים אוטומטית

### מצב כהה:

- לחץ על כפתור הירח/שמש בכותרת
- ההעדפה נשמרת אוטומטית

## 🔧 התקנה כאפליקציה (PWA)

האפליקציה מוכנה להתקנה כפרוגרסיבית ווב אפ (PWA) עם תמיכה מלאה ב-offline.

### 📱 הוראות התקנה מפורטות

**להנחיות מפורטות להתקנה על Android ו-iOS, ראה את הקובץ [INSTALL.md](INSTALL.md)**

### התקנה מהירה:

#### Android (Chrome):
1. פתח את האפליקציה ב-Chrome
2. לחץ על תפריט (⋮) > "הוסף למסך הבית" או "Install app"
3. האפליקציה תותקן ותופיע במסך הבית

#### iOS (Safari):
1. פתח את האפליקציה ב-Safari
2. לחץ על כפתור השיתוף (□↑)
3. בחר "הוסף למסך הבית"

#### מחשב (Chrome/Edge):
1. פתח את האפליקציה בדפדפן
2. לחץ על סמל ההתקנה (➕) בשורת הכתובת
3. האפליקציה תותקן ותופיע בתפריט ההתחלה

### תכונות PWA:

- ✅ **עבודה offline מלאה** - כל התכונות עובדות ללא חיבור לאינטרנט
- ✅ **מהירות גבוהה** - טעינה מהירה בזכות מטמון מקומי
- ✅ **מצב standalone** - נראה כמו אפליקציה רגילה (ללא שורת כתובת)
- ✅ **עדכונים אוטומטיים** - האפליקציה מתעדכנת בעצמה
- ✅ **אייקון במסך הבית** - גישה מהירה

```
Shopping/

├── icon-192.png       # אייקון 192x192 (צריך להוסיף)
├── icon-512.png       # אייקון 512x512 (צריך להוסיף)
└── README.md          # קובץ זה
```

## 🎨 יצירת אייקונים

האפליקציה דורשת אייקונים להתקנה כ-PWA:

### שיטה 1: יצירה אוטומטית (מומלץ)

1. פתח את הקובץ `generate-icons.html` בדפדפן
2. לחץ על "הורד אייקון 192x192" ו-"הורד אייקון 512x512"
3. שמור את הקבצים בתיקיית הפרויקט

### שיטה 2: יצירה עם Node.js

```bash
npm install canvas
node generate-icons.js
```

### שיטה 3: יצירה ידנית

1. צור שני קבצי PNG:
   - `icon-192.png` - בגודל 192x192 פיקסלים
   - `icon-512.png` - בגודל 512x512 פיקסלים

2. אפשר להשתמש בכלי כמו:
   - [Favicon Generator](https://favicon.io/)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)
   - כל עורך תמונות (Photoshop, GIMP, וכו')

3. שמור את הקבצים בתיקיית הפרויקט

**הערה:** האפליקציה תעבוד גם בלי האייקונים, אבל הם נדרשים להתקנה כ-PWA.

- מצב כהה/בהיר

## 🐛 פתרון בעיות

### האפליקציה לא נטענת:
- ודא שכל הקבצים נמצאים באותה תיקייה
- פתח את קונסול הדפדפן (F12) לבדיקת שגיאות
- נסה לפתוח בדפדפן אחר

### הנתונים לא נשמרים:
- ודא שהדפדפן מאפשר LocalStorage
- בדוק שהדפדפן לא במצב "פרטי" או "גלישה בסתר"
- נסה לנקות את המטמון ולהתקין מחדש

### PWA לא עובד:
- **חובה להשתמש ב-HTTP/HTTPS** (לא file://)
- השתמש בשרת מקומי (ראה הוראות לעיל)
- ודא שה-manifest.json ו-sw.js נטענים בהצלחה
- בדוק ב-Developer Tools > Application > Service Workers

### האפליקציה לא עובדת offline:
- ודא שה-Service Worker נרשם (Developer Tools > Application > Service Workers)
- נסה לטעון מחדש את הדף
- נקה את המטמון והתקן מחדש

### התקנה לא עובדת:
- **Android**: ודא שאתה משתמש ב-Chrome
- **iOS**: ודא שאתה משתמש ב-Safari (לא Chrome)
- ראה הוראות מפורטות ב-[INSTALL.md](INSTALL.md)
