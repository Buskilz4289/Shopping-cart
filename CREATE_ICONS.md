# יצירת איקונים לאפליקציה

## שיטה 1: שימוש ב-generate-icons.html (מומלץ - הכי פשוט)

1. פתח את הקובץ `generate-icons.html` בדפדפן
2. לחץ על "הורד אייקון 192x192"
3. לחץ על "הורד אייקון 512x512"
4. שמור את הקבצים בתיקיית הפרויקט בשמות:
   - `icon-192.png`
   - `icon-512.png`

## שיטה 2: שימוש ב-generate-icons.js (דורש Node.js)

1. התקן את החבילה הנדרשת:
   ```bash
   npm install canvas
   ```

2. הרץ את הסקריפט:
   ```bash
   node generate-icons.js
   ```

3. הקבצים `icon-192.png` ו-`icon-512.png` ייווצרו אוטומטית

## שיטה 3: יצירה ידנית

אם אתה רוצה ליצור איקונים מותאמים אישית:

1. צור תמונות PNG בגדלים:
   - 192x192 פיקסלים
   - 512x512 פיקסלים

2. שמור אותן בשמות:
   - `icon-192.png`
   - `icon-512.png`

3. מקם את הקבצים בתיקיית הפרויקט (באותה תיקייה כמו `index.html`)

## הערות

- האיקונים משמשים ל:
  - התקנת האפליקציה כמתקנת (PWA)
  - תצוגה במסך הבית
  - תצוגה ברשימת האפליקציות

- אם האיקונים לא קיימים, האפליקציה עדיין תעבוד, אבל לא תוכל להתקין אותה כמתקנת.
